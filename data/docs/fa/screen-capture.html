Amped Screen Capture Documentation

Why a Screen Capture tool

One of the biggest and most time consuming issues working on CCTV videos is that quite often the files are encoded in proprietary video formats. To play these files you need specific players provided by the system producer. If you want to analyze and process these videos, it is necessary to convert them to a standard format. Unfortunately, most of the time, player software is not properly designed for investigative and forensics usage, so an export function is often not provided or; if it there, it causes further reduction in the video quality. In these cases, capturing screen output played from the native player is the only way to work with it. While several screen capture tools are available, the Screen Capture utility integrated in FIVE allows the user to easily save to uncompressed video, a very important requirement for forensic applications, and to have control over the capture process. At the end of the capture, the resulting video file is open in FIVE and you are ready to get to work.

Layout of the Screen Capture tool

  1. Output File: This is a location that you set for the new video file. The default location is Desktop, the default file name is capture followed by a timestamp to avoid overwriting. 2- Output Format: The output format is the format for the new video that is created by the capture. The default is avi-Raw Video, which makes a large file, but is the highest quality. The other formats can also be used depending on your needs and the tradeoff between disk usage, quality and codec compatibility. Capture Region: Choose between full screen and a specific area, such as the player screen only.
  2. Capture Frame Rate: The default is 10 fps, but you will need to adjust it in order to reflect, if possible, original video frame rate.

Using the Screen Capture Utility:

  • Open the proprietary file in either the DVR console or in the proprietary player. It is important to capture the video in its most original formatted size and frame rate. First, try to set the window size in the player or DVR console to the default original playback video size (no full screen or custom resizing of the window, as this will cause unwanted pixel interpolation).

  • Open the Amped Screen Capture utility for FIVE, clicking on File / DVR Screen Capture...:

  • Make any changes necessary to the basic settings for Amped Screen Capture. It is very important that you set the correct frame rate for capture the video. If the capture frame rate is too low you will lose some frames. If it is too high, you will have duplicate frames which will complicate the further analysis on the video. Even though FIVE has the Remove Duplicates function available, it is still better to get it right as early in the process as possible. If the frame rate is constant (a condition not always apparent in surveillance videos), you can simply look at the video timestamp and see how often the seconds text changes. If it changes every 10 frames, for example, we'll set 10 frames per seconds for our capture. Remember that you will need to capture the video closest to the original speed to get the capture right.

  • In this case, we only want to capture the area of the player screen only. So in Capture Region, we choose Select Region:

  • Your monitor screen will fade and a box selection tool will appear with the mouse pointer. You then select an outline of the screen area. In this case, it is partially obstructed by the Amped Screen Capture tool, but we can easily move it out of the way when we need to.

  • Now the points of the area of the screen are plotted so this particular area is the area captured. If you have made a mistake on the area, reselect the Capture Region and re-plot your capture area rectangle.

  • Now you can start the capture using the Start Capture button.

  • Now press play on the proprietary player:

  • When you have finished playing and simultaneously capturing the part or whole video that you need to capture, hit the Stop button to stop the capture. You can also shut down the proprietary player.

  • The file should now open in Windows Media Player (or your default media player):

  • If the quality or smoothness of play is not what you would like it to be, you can redo the process until you achieve the results you need.

  • If the frame rate for the capture window were set too high, or the capture window was set too large (such as fullscreen) you may notice several lost frames. In this case you will need to recapture the video and either reduce the capture area size or the frame rate. A good trick, if allowed by your player, is to reduce the playback speed of the video and then adjust the capture frame rate accordingly.

  • Once you close the Screen Capture window, the captured files will be automatically loaded in FIVE in your current project.
