Tutorials

In this section we describe how to solve some common problems related to the image quality when using Amped FIVE. Under the name of the tutorial there is a short summary of what the reader will learn. Please note that you can find all the files in the examples in the folder Samples of the Amped FIVE installation directory or, having Amped FIVE open, by clicking on Help → Open Samples Projects Folder.

  1. Deblurring a license plate: in this tutorial we will learn how to open an image file, use the Motion Deblurring filter and generate a report file.
  2. Enhancing a dark license plate: in this tutorial we will learn how to enhance a dark license plate in a video file using the Exposure filter, stabilizing it using the Local Stabilization filter, reducing the noise with Frame Averaging and modifying a filter while showing the impact it has on the others.
  3. Enhancing a face: in this tutorial we will learn how to enhance a person behind a window glass deinterlacing the video, selecting the portion of interest, stabilizing the character's face with Local Stabilization, getting a better result with Frame Averaging, enhancing the edges with Unsharp Masking
  4. Measuring the height of a subject: in this tutorial we will learn how to measure the height of a person in a crime scene using Measure 3d.
  5. Separate a latent fingerprint from the background: in this tutorial we will learn how to remove the fine-line security pattern on a banknote to enhance a fingerprint using the Fourier transform filter.
  6. Correct the perspective of a license plate: in this tutorial we will learn how to get a better license plate if affected a perspective distortion.
  7. Screen capture of a DVR player: in this tutorial we will learn how to capture the screen while playing a DVR file in its native player.
  8. Perspective Stabilization and Perspective Super Resolution: in this tutorial we will learn how to combine the Perspective Stabilization and Perspective Super Resolution filters to obtain, from a sequence with several frames, a single output image with better quality.
  9. Perspective Registration: in this tutorial we will learn how to obtain a higher-quality image of a planar object (e.g. a license plate) starting from multiple low-quality (noisy) images of the same object taken from different points of view.
